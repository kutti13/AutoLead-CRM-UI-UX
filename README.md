# 🚗 AutoLead CRM - UI/UX Design

## 📌 Overview
AutoLead CRM is a Lead Management application designed for HSR Motors using Figma. It helps sales teams manage customer leads efficiently through an intuitive interface.

## ✨ Features
- Lead Listing
- Lead Details
- Dashboard
- Search Functionality
- Lead Status Tracking
- Lead Source Tracking
- Recent Activities

## 🛠 Tools Used
- Figma
- UI/UX Design
- Wireframing
- Prototyping
